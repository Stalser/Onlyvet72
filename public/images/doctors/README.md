# Фотографии врачей

В эту папку нужно добавить фотографии врачей в формате JPG или PNG.

## Требования к фото:
- **Формат**: JPG или PNG
- **Размер**: минимум 400x400px (квадратное)
- **Имя файла**: `elvin.jpg`, `diana.jpg`, `oleg.jpg`

## Как добавить:
1. Поместите фото врачей в эту папку (`public/images/doctors/`)
2. Назовите файлы соответственно ID врача:
   - `elvin.jpg` — для Эльвина Мазагировича
   - `diana.jpg` — для Дианы Чемериловой
   - `oleg.jpg` — для Олега Врача
3. Сайт автоматически подхватит фото

Если фото не добавлены, будут отображаться аватарки с инициалами.
