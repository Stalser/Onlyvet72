# OnlyVet — Сайт ветеринарных онлайн-консультаций

## 📁 Структура проекта

```
onlyvet/
├── src/
│   ├── app/                      # Страницы сайта (Next.js App Router)
│   │   ├── layout.tsx            # Общий layout с Header и Footer
│   │   ├── page.tsx              # Главная страница
│   │   ├── booking/              # Форма записи на консультацию
│   │   ├── services/             # Услуги (с фильтром и поиском)
│   │   ├── doctors/              # Врачи (список + страницы врачей)
│   │   ├── prices/               # Цены (с фильтром, сортировкой, сравнением)
│   │   ├── how-it-works/         # Как проходит консультация
│   │   ├── safety/               # Безопасность
│   │   ├── reviews/              # Отзывы
│   │   ├── faq/                  # FAQ
│   │   ├── about/                # О сервисе
│   │   └── documents/            # Документы
│   ├── components/               # React компоненты
│   │   ├── Header.tsx            # Шапка с навигацией, поиском и выпадающим меню
│   │   ├── Footer.tsx            # Подвал
│   │   ├── ServiceCard.tsx       # Карточка услуги
│   │   ├── DoctorCard.tsx        # Карточка врача (с поддержкой фото)
│   │   └── ReviewCard.tsx        # Карточка отзыва
│   └── data/                     # Данные (услуги, врачи, отзывы)
│       ├── services.ts           # Список услуг с ценами
│       ├── doctors.ts            # Список врачей
│       └── reviews.ts            # Отзывы
├── public/
│   ├── images/
│   │   ├── doctors/              # Фото врачей (добавить вручную)
│   │   ├── logo.svg              # Логотип OnlyVet
│   │   ├── logo-header.png       # Логотип для шапки
│   │   └── logo.png              # Логотип PNG
│   └── favicon.ico               # Favicon
├── Новое/                        # Исходные файлы (логотипы, документы)
└── package.json
```

## 🎨 Дизайн

### Цветовая палитра (из логотипа)
- **Primary**: `#283C63` — тёмно-синий (основной цвет бренда)
- **Secondary**: `#928A97` — серо-фиолетовый (цвет сердечка)
- **Primary Light**: `#E8ECF3` — светлый фон
- **Background**: `#FAFBFC` — основной фон

### Стиль
- Медицинский, спокойный, минималистичный
- Много воздуха между блоками
- Плавные переходы и hover-эффекты
- Адаптивная вёрстка (mobile-first)
- Выпадающее меню в навигации
- Поиск по сайту

## 🚀 Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен-сервера
npm start

# Линтинг
npm run lint
```

Сайт доступен по адресу: http://localhost:3000

## 📄 Страницы

### Основные
- **Главная** (`/`) — витрина со всеми блоками
- **Услуги** (`/services`) — список всех услуг с фильтром и поиском
- **Врачи** (`/doctors`) — список всех врачей с фильтром по специализации
- **Врач** (`/doctors/[id]`) — страница конкретного врача
  - `/doctors/elvin` — Эльвин Мазагирович
  - `/doctors/diana` — Диана Чемерилова
  - `/doctors/oleg` — Олег Врач
- **Цены** (`/prices`) — прайс-лист с фильтром, сортировкой и сравнением
- **Как это работает** (`/how-it-works`) — процесс консультации
- **Безопасность** (`/safety`) — ограничения и возможности онлайн-формата
- **Отзывы** (`/reviews`) — отзывы клиентов
- **FAQ** (`/faq`) — часто задаваемые вопросы
- **О сервисе** (`/about`) — информация о проекте

### Форма записи
- **Запись на консультацию** (`/booking`) — форма с выбором формата
  - Поддержка выбора врача: `?doctorId=elvin`
  - Поддержка выбора услуги: `?serviceId=online-consult`

### Документы
- **Документы** (`/documents`) — список всех документов
- **Пользовательское соглашение** (`/documents/terms`)
- **Политика конфиденциальности** (`/documents/privacy`)
- **Правила оплаты и возврата** (`/documents/refund`)
- **Согласие на консультацию** (`/documents/consent`)
- **Дисклеймер** (`/documents/disclaimer`)
- **Политика cookies** (`/documents/cookies`)
- **Правила сайта** (`/documents/site-rules`)

- Фильтр по категории (консультация, второе мнение, диагностика, сопровождение)

- Поиск по названию, описанию, тегам

### Врачи
- Фильтр по специализации

### Цены
- Фильтр по категории
- Сортировка (по цене, по названию)
- Поиск по названию, описанию, тегам
- Таблица сравнения всех услуг

## 📷 Фото врачей

Для добавления фото врачей:
1. Поместите фото в `public/images/doctors/`
2. Назовите файлы: `elvin.jpg`, `diana.jpg`, `oleg.jpg`
3. Рекомендуемый размер: 400x400px (квадрат)

Если фото не добавлены, отображаются аватарки с инициалами.

## 📝 Что нужно сделать

1. **Заменить тексты документов** на актуальные из файлов `.pages` (папка `Новое/`)
2. **Добавить фото врачей** в `public/images/doctors/`
3. **Настроить отправку форм** (сейчас данные выводятся в консоль)
4. **Настроить аналитику** (Яндекс.Метрика, Google Analytics)
5. **Подключить домен** и SSL-сертификат
